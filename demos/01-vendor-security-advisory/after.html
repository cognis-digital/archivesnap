Security Advisory ACME-SA-2026-014

Summary

An authentication-bypass vulnerability affects the Acme Appliance management web interface on releases 4.2 through 4.5. A patched release is now available and customers should upgrade promptly.

Status

Severity: High

Fixed versions: 4.5.1 and 4.6.0. A patch is now available from the customer portal.

Workarounds

Restrict access to the management interface to trusted networks and rotate administrative credentials after upgrading.
